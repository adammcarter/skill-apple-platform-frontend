# Example Prompts

Use these as starting points when invoking `$apple-platform-frontend`.

## New Screen

`Use $apple-platform-frontend to build a macOS inspector sidebar in SwiftUI for image metadata editing. It should feel native, support keyboard-first use, use SF Symbols where appropriate, and include previews for empty, populated, and disabled states.`

## Refactor

`Use $apple-platform-frontend to refactor this iOS detail screen into clearer SwiftUI view composition. Keep the existing behavior, improve hierarchy and spacing, and make loading, error, and empty states explicit.`

## Review

`Use $apple-platform-frontend to review this Apple-platform UI change for platform fit, accessibility, motion, iconography, and missing state coverage. Findings first, with file references.`

## Accessibility Pass

`Use $apple-platform-frontend to improve this SwiftUI form for accessibility. Check labels, values, keyboard navigation, Dynamic Type resilience, contrast, reduced motion, and VoiceOver semantics, then implement the fixes.`

## Motion Tuning

`Use $apple-platform-frontend to improve the transitions in this screen so they communicate insertion, removal, and selection more clearly. Keep them restrained and respect reduced motion.`
